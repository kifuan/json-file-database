Interface Collection<T, P>

A collection is like an array. You can insert, update, delete and find elements in it.

If you called methods that affect the collection, it will start a so-called "debounced" function to save the data.

Type Parameters

  • T extends object

    the type of element.

  • P extends keyof T

    the prime key for element type.
